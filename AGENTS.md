# Product Requirements Document (PRD)
## 프로젝트명: RunLoop (가칭) - 위치 기반 랜덤 순환 러닝 코스 추천 서비스

### 1. 서비스 개요 및 목표
RunLoop는 사용자의 현재 위치를 기반으로 원하는 반경과 거리에 맞춰 '원점 회귀(Loop)' 형태의 새로운 러닝 코스를 랜덤으로 추천해 주고, 실제 러닝 트래킹 및 기록 분석을 제공하는 모바일 애플리케이션이다. 매일 같은 코스를 뛰는 러너들에게 새로운 자극과 탐험의 재미를 제공한다.

### 2. 타겟 유저
- 러닝에 관심 있는 전 연령층 (특히 새로운 러닝 스팟을 찾고 기록을 인증하기 좋아하는 2030 유저)

### 3. 주요 기능 요구사항 (Requirements)

#### 3.1. 인증 및 사용자 관리
- 카카오톡 OAuth를 이용한 최초 로그인 및 회원가입 기능 (Supabase Auth 활용).

#### 3.2. 코스 생성 옵션 입력
- **기준 좌표:** 현재 유저의 기기 GPS 위치 또는 사용자가 직접 입력한 주소.
- **반경 설정:** 500m, 1km, 5km (라디오 버튼 또는 세그먼트 컨트롤).
- **목표 거리 설정:** 최소 1km ~ 최대 10km (Slider UI 활용, Step: 1km).

#### 3.3. 안전한 랜덤 러닝 코스 생성 및 제공 (Core)
- 입력된 기준 좌표, 반경, 목표 거리를 바탕으로 **시작점과 종료점이 동일한 순환형(Loop) 러닝 코스**를 랜덤 생성.
- **[중요 - 안전 라우팅]:** GIS 라우팅 API(Mapbox 또는 OpenRouteService 등) 호출 시, 반드시 **보행자 전용(Pedestrian) 프로필**을 사용해야 하며, **자동차 전용 도로나 비포장도로(Unpaved/Dirt)는 경로 생성에서 철저히 배제**해야 한다.
- 코스의 총 거리 및 예상 소요 시간, 고도(Elevation) 정보를 함께 제공.

#### 3.4. 코스 승낙 및 재탐색
- 제안된 코스에 대해 "이 코스로 달리기(승낙)" 또는 "다시 생성하기(재시도)" 기능 제공.

#### 3.5. 실시간 러닝 트래킹
- "달리기 시작" 시 기기의 백그라운드 GPS를 활성화하여 유저의 실시간 위치를 추적.
- 지도 상에 추천된 '목표 코스'와 사용자가 '실제 달린 궤적'을 실시간으로 겹쳐서 비교 표시.
- 러닝 중 페이스, 경과 시간, 달린 거리 등을 Glanceable한 대형 타이포그래피 UI로 제공.

#### 3.6. 러닝 결과 저장 및 공유
- 러닝 완료 시 실제 달린 거리, 소요 시간, 페이스, 경로(GeoJSON 등) 데이터를 DB에 저장.

#### 3.7. 러닝 히스토리 및 3D/2D 분석 뷰 (Single & Multi-View)
- 마이페이지에서 과거의 러닝 기록 리스트 제공.
- **단일 히스토리 모드:** 특정 러닝 기록 선택 시 해당 코스만의 궤적과 상세 분석 데이터를 3D/2D 맵에 시각화.
- **다중 히스토리 모드 (Multi-View):** 여러 개의 러닝 기록을 동시에 지도에 띄워 누적된 러닝 발자취를 확인하는 기능.
  - **[UX 제약사항]:** 다중 히스토리 뷰 활성화 시, 각기 다른 날짜/시간에 달린 코스 궤적들을 지도 위에서 시각적으로 명확히 구분할 수 있도록 **각 궤적마다 고유한 색상(Unique Color) 또는 뚜렷한 패턴/마커를 부여하는 UI/UX 로직**이 반드시 포함되어야 한다.

### 4. 기술 스택 (Tech Stack)
- **Package Manager:** **pnpm** (npm, yarn 사용 금지)
- **Frontend:** React Native (Expo)
- **Styling:** NativeWind (Tailwind CSS) - DX 및 유지보수 최적화
- **State Management:** Zustand (클라이언트 상태) + React Query (서버 상태)
- **Backend & DB:** Supabase (PostgreSQL + PostGIS) - 카카오 로그인 및 GIS 쿼리 처리
- **Map & Routing:** react-native-maps, Mapbox API (라우팅, 3D Map, 보행자/도로 속성 필터링)
